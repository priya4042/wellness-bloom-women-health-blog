## Local Setup

1. Open terminal in project root.
2. Run:
   ```bash
   npm install
   npm run build:content
   npm run validate:links
   ```
3. Serve with any static server (example):
   ```bash
   npx serve .
   ```

## Deploy (Simple)

### Option A: Netlify
1. Push this project to GitHub.
2. In Netlify: New site from Git.
3. Build command: leave empty.
4. Publish directory: `.`
5. Deploy.

### Option B: Vercel
1. Push to GitHub.
2. Import repository in Vercel.
3. Framework preset: Other.
4. Build command: empty.
5. Output directory: `.`
6. Deploy.

### Option C: GitHub Pages
1. Push repository to GitHub.
2. Repository Settings -> Pages.
3. Source: Deploy from branch.
4. Branch: `main` and folder `/ (root)`.
5. Save.

## GitHub Setup Guide

1. Create a new GitHub repository.
2. Push project:
   ```bash
   git init
   git add .
   git commit -m "Initial production-ready blog"
   git branch -M main
   git remote add origin https://github.com/<your-user>/<your-repo>.git
   git push -u origin main
   ```
3. Confirm GitHub Actions are enabled (Repository -> Actions).

## Automation Setup (Free Daily Posting)

- Workflow file: `.github/workflows/daily-post.yml`
- Trigger: every day via cron + manual trigger.
- What it does:
  - Builds baseline content
  - Adds one new daily post
  - Validates links
  - Commits and pushes changes automatically

### To test automation manually
1. Open GitHub -> Actions -> Daily Blog Automation.
2. Click Run workflow.
3. Confirm new post appears in:
   - `data/posts.json`
   - `blog/<new-slug>/index.html`
   - `sitemap.xml`

## AdSense Integration Steps

1. Replace ad placeholders in HTML templates where `AdSense ... Placeholder` is shown.
2. Add your AdSense script in each page `<head>` (or inject via template tooling):
   ```html
   <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXXXX" crossorigin="anonymous"></script>
   ```
3. Replace `ads.txt` publisher ID:
   - `pub-XXXXXXXXXXXXXXXX` -> your real publisher ID.
4. Keep placements in:
   - Header ad block
   - Sidebar ad block
   - In-content ad blocks
   - Footer ad block

## SEO Setup Checklist

1. Replace `https://example.com` with your real domain in:
   - all canonical tags
   - Open Graph URLs
   - `robots.txt`
   - `sitemap.xml`
2. Regenerate content after domain changes if needed:
   ```bash
   npm run build:content
   ```
3. Submit sitemap to Google Search Console.
4. Keep daily automation enabled so fresh content is indexed regularly.

## Required Scripts

- `npm run build:content` -> Generates all posts + sitemap + static post pages.
- `npm run auto:post` -> Adds one new post for today.
- `npm run validate:links` -> Checks missing local href/src targets.
